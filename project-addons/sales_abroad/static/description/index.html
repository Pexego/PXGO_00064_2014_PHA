Sales abroad documents for Odoo

Show documents required for sales abroad

Features:
1. Assignment of documents to sales by destination country
2. Show confirmation dialog for those sales to countries requiring additional documents
3. If system variable sales_abroad_directory is defined, save a copy of attachments with original filename (plus attachment id) in the directory specified under a folder with country name, for easy access to documents
